HTML TAGS

HTML tags are the building blocks of HTML documents. They are used to create elements on a webpage, such as headings, paragraphs, links, images, and more. Each tag has a specific purpose and is enclosed in angle brackets(< >).


COMMON HTML TAGS


  • Structure

  • Text & content

  • Lists

  • Links & multimedia

  • Forms & input

  • Layout & containers

  • Interactive & grouping

  • Tables

  • Grouping content

  • Meta tags(hidden tags)

  • Examples

1.STRUCTURE

TAGS(< >) USE
!DOCTYPE html Shows that it is HTML5 document
html The root element
head contains the metadata,title and CSS links
body Contains all the visible content

2.TEXT & CONTENT

TAGS(< >) USE
h1-h6 Headings, h1 being the biggest & h6 the smallest
p Defines paragraphs
br Line break
strong BOLD text
em Italic text
u Underline text

3.LISTS

TAGS(< >) USE
ul Unordered list (bulleted)
ol Ordered list (numbered)
li List item, used inside ul or ol

4.LINKS & MULTIMEDIA

TAGS(< >) USE
a Defines a hyperlink (use href attribute)
img Embeds an image(use src & alt attribute)
video Embeds a video
audio Embeds audio files

5.FORMS & INPUT

TAGS(< >) USE
form A container for user input
input Defines a single-line input field (use type attribute for different input types like text,password,email,url...)
button Defines a clickable button
progress A progress bar
text area Multi line text box

6.Layout & Containers

TAGS(< >) USE
div A generic container for grouping HTML elements
span A generic container for small bits of text inside a paragraph.
header,footer,main,section,nav,aside,article Semantic tags that help screen readers & search engines understand the page structure.

7.INTERACTIVE & GROUPING

TAGS(< >) USE
select Create a dropdown list.
option An item inside select dropdown
fieldset Groups related elements in a form
legend Defines a caption for a fieldset
details Create a toggle widget that the user can open & close
summary The visible heading for a details

8.TABLES

TAGS(< >) USE
table Defines an HTML table
tr Defines a row in an HTML table
td Defines a cell in an HTML table
th Defines a header cell in an HTML table

9.Grouping Content

TAGS(< >) USE
hr A horizontal break line
figure Used to wrap a diagram or code snippet.
figcaption A caption for the content inside figure

10.Meta Tags(Hidden Tags)

TAGS(< >) USE
meta Provides metadata about the HTML document (use name and content attributes)
link Used to link external resources like CSS files (use rel and href attributes)
script Used to link external resources like Javascript files
title Defines the title of the document, shown in the browser tab
meta charset="UTF-8" Tells the browser which character set to use
